## 回答要求

基于以上检索到的医学知识和用户信息，请按照系统提示词的要求生成回答。

### 特别注意
1. **安全第一**: 如果检索结果不足以安全回答，或者用户描述严重症状，必须触发安全兜底
2. **准确引用**: 所有关键信息必须标注来源文档编号
3. **结构完整**: 遵循系统提示词中定义的回答结构
4. **专业适度**: 使用专业但易于理解的医学语言
5. **人文关怀**: 体现对患者的理解和关心

## 回答生成指令

请生成完整的医疗知识回答，包含以下部分：

1. **回答标题**: 简洁概括回答的核心内容
2. **核心摘要**: 2-3句概括性回答
3. **详细解释**: 基于检索结果的分点详细解释
4. **注意事项**: 相关的注意事项和警告
5. **医疗建议等级**: 自我护理、就医指征、紧急情况
6. **信息来源**: 明确标注引用来源

## 模板变量说明

- `{query}`: 用户原始查询
- `{timestamp}`: 查询时间戳
- `{context}`: 用户对话上下文（如果有）
- `{documents}`: 检索到的文档片段，格式如下：
  ```
  [文档1] 标题: {title}
  相关性: {score}
  内容: {content}
  
  [文档2] 标题: {title}
  相关性: {score}
  内容: {content}
  ...
  ```
- `{doc_count}`: 检索到的文档数量
- `{retrieval_mode}`: 使用的检索模式（向量/关键词/混合）
- `{retrieval_time_ms}`: 检索耗时（毫秒）
- `{rerank_enabled}`: 是否启用了重排序
- `{reranker_model}`: 使用的重排序模型
- `{health_profile}`: 用户健康档案信息，如果可用
